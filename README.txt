MonitorHead v1.0.0
==================

Установка:
1. Распакуйте все файлы в папку по вашему выбору. Если не менять ничего, то
      программа установится по следующему пути: C:\Program Files\MonitorHead\.
2. Запустите MonitorHead.exe:
     - из папки C:\Program Files\MonitorHead\;
     - ярлык на рабочем столе (если при установке был выбран пункт "создать 
          ярлык на рабочем столе").

Требования:
- Windows 10/11 (64-бит)
- Дополнительная установка не требуется.

Лицензия:
Это приложение использует библиотеку Qt под лицензией LGPL v3.
Полную информацию о лицензии см. в папке «licenses».
